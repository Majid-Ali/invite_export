
-- SUMMARY --

This module export brief statics of inviter and invitee into an xls file. 
For a full description visit the project page:
  http://drupal.org/project/1848512
Bug reports, feature suggestions and latest developments:
  http://drupal.org/project/issues/1848512


-- REQUIREMENTS --

* Token module http://drupal.org/project/invite


-- INSTALLATION --

1. Copy the invite module to your modules directory and enable it on the Modules
   page (admin/build/modules).





-- USAGE --

Menu item will appear on the admin navigation click on it to generate a xls sheet and download it.
It will contain some basic statictis of invitee and inviter such as how was the inviter, number of rcords will be displayd for each invitation sent by  inviter. Invitee status about if he joined the site or not etc. Please if you find any bug or have any suggestion to inmprove this module write me an email.
